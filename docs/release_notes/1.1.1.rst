 * `本来ならある設定が足りない場合に、毎回ファイルに書き込んで、最大10回書き込んでたのを修正 <https://github.com/BlackListener/BlackListener/pull/6/files#diff-168726dbe96b3ce427e7fedce31bb0bcR248>`_
